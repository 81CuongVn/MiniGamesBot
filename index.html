MiniGamesBot

A Python Discord bot with different kinds of minigames.

You can play minigames against a friend, the bot or alone.
All games are easy to use with clicking on reactions.
The command prefix is "?".



MiniGames

blackjack
Start a game of blackjack against the bot
scramble
Start a game of scramble and try to put the letters in the right order
connect4 [@player1] [@player2]
Start a game of connect4 against a friend
hangman
Start a game of hangman
guessword
Start a guessword minigame and try and guess the word from it's given definition
quiz
Start a quiz in a category of choice


Other

info
Displays some info about the bot
help
Gives a list of all commands
